Biography

Basic Information

Name Bonan Yan
Date of Birth June 27th, 1991
Languages Mandarin, English
中文简介 燕博南，北京大学人工智能研究院助理教授（副高级），北京大学小米博雅青年学者，博士生导师，北京大学通用人工智能芯片研究中心执行主任。2020年于杜克大学获得电子与计算机工程博士学位。研究领域包括：通用人工智能体处理器芯片设计、SRAM与新兴存储器(MRAM、RRAM、PCM、FeFET等)集成电路设计，已于国际知名期刊和会议上发表论文50余篇，包括Nature Electronics、Nature Communications、ISSCC、IEDM等，荣获ACM SIGDA Outstanding New Faculty Award、中国计算机学会CCF-华为胡杨林基金存储领域专项优秀方案奖等，担任中国能源研究会电力传感和智能分析专业委员会第一届委员、DAC/DATE/EDTM等多个世界高水平学术会议技术委员会成员，研发的数字型SRAM存内计算智能计算处理器芯片刷新当时存算一体计算效率世界纪录、提出"通用伊辛机"概念与组成原理、提出"认知处理单元"概念与组成原理。作为项目负责人承担国家自然科学基金委原创探索项目、国家自然科学基金委重大研究计划（培育）项目、北京大学本科教学改革项目，入选国家级高层次青年人才项目、北京市科技新星计划(创新新星)、北京大学未名青年学者人才资助计划。
